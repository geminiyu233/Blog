# blog

> 个人博客客户端

## 技术栈

``` bash
Nuxt + Vuex + SaaS + Axios + Koa2 + Mongoose
```

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```

## 功能列表

- [ ] 个人信息展示

- [ ] 推荐列表

- [ ] 归档（按时间）

- [ ] 归档（标签）

- [ ] 文章关键词搜索
